git 저장소 상태로 만드는 방법 : cd 저장소 이름

저장소의 상태를 보는 명령어 git status
파일을 추적하게 하려면... : git add 파일이름 (commit 상태로 바꾸는 거)
commit 된 상태에서 git push 하고, 토큰 인증코드를 집어넣으면 git에 올라감.

브렌치(branch)
- 게임 세이브로 현재 상태를 저장할 수 있다. 
- 저장 상태 후 플레이 도중 이전 상태로 되돌리고 싶을 경우 세이브를 로드 
- 원하는 시점을 로드하여 확인이나 수정 가능 
- 이때, 현재 상태 저장은 사용자의 선택 등을 저장하지 않는다. 

브렌치를 통한 파일 별도 보관 
- 저장소의 기본 브렌치 : master 또는 main
- 현재 저장소의 상태
